# Workflows

Export N8N workflows as JSON and save here.

## Naming Convention

| File | Workflow |
|---|---|
| `workflow_daily_scanner.json` | Workflow 1 — Daily VPS Scanner |
| `workflow_cve_analysis.json` | Workflow 2 — CVE Analysis Pipeline |
| `workflow_hitl_handler.json` | Workflow 3 — HITL Handler |
| `workflow_alert_system.json` | Workflow 4 — Alert System |
| `workflow_weekly_summary.json` | Workflow 5 — Weekly Summary |

## How to Export from N8N

1. Open workflow in N8N
2. Click three dots menu (top right)
3. Download → JSON
4. Save here with correct name
